## Sobre o projeto
O **StreamFlix** é um site de streaming fictício que simula a interface de um serviço como a Netflix.  
O objetivo é demonstrar habilidades em **HTML e CSS**, além de design responsivo com **adaptação para celular**.

## Tecnologias utilizadas
- HTML5
- CSS3 (Flexbox, Grid)
- Design responsivo para diferentes telas
